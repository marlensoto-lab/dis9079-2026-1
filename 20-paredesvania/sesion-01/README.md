## Protocolos

- **MQTT** (Message Queuing Telemetry Transport): Protocolo de transporte de mensajería, eficiente y diseñado para redes inestables o de bajo ancho de banda. Utiliza un agente **(broker)** para gestionar publicaciones y suscripciones, siendo más eficiente que HTTP para dispositivos con recursos limitados. 
- **HTTPS** (HyperText Transfer Protocol Secure): Protocolo estándar para la transferencia de documentos web, utilizado cuando se requiere seguridad y la carga de datos no es crítica en tiempo real. 
- **OSC** (Open Sound Control): Protocolo para el intercambio de información entre computadoras, sintetizadores de sonido y otros dispositivos multimedia. Se destaca por su alta velocidad y precisión, ideal para rendimiento en directo, a diferencia de la latencia que puede tener MQTT.
