### ¿Qué significa...? (lo que alcancé a entender)

void setup () = inicio

void loop () = bucle hasta que explote o lo apagues

{ texto } = lo que hay dentro de los murciélagos es lo que pasa

serial = cable USB

// texto texto texto = comunicación en Arduino, da lo mismo si hay un espacio entre los // y el texto, pero se puede añadir por respeto y fineza

#textotextotexto = comunicación en mosquitto

(- c) = configuración

(- h) = ayuda

Pantalla.begin() = el "." significa "dentro de"
