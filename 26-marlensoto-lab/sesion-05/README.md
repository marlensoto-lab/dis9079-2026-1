## Solemne 1 ##
Durante esta actividad realizamos la configuración inicial de los dispositivos y del entorno de trabajo.

En primer lugar, reiniciamos los Arduino para asegurar un funcionamiento limpio y evitar errores previos.

Posteriormente, instalamos la biblioteca más reciente de Adafruit IO Arduino desde el entorno de Arduino IDE, lo que nos permitió establecer la conexión con la plataforma Adafruit IO.

Para trabajar con la plataforma, utilizamos credenciales personales (usuario y clave), las cuales se integran directamente en el código para permitir la conexión entre el dispositivo y Adafruit IO.

Como parte del ejercicio, realizamos la modificación del feed en Adafruit IO, con el objetivo de enviar datos desde la plataforma hacia el Arduino. Esto implicó ajustar el código para que el dispositivo pudiera recibir correctamente la información.

Finalmente, enviamos el código modificado al Arduino, verificando su funcionamiento mediante la recepción de datos desde Adafruit IO.
