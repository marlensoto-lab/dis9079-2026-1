En la clase se presentó el curso, el equipo docente y el uso de GitHub como bitácora de trabajo. El curso se enfoca en la relación entre computación física, diseño e interacciones inalámbricas.

Se trabajará con microcontroladores como el Arduino Uno R4 WiFi y el Raspberry Pi Pico 2W, utilizando computadores personales para configurar entornos de desarrollo.

## Protocolos y comunicación

Se revisaron protocolos fundamentales para la comunicación entre dispositivos:
+ HTTPS: comunicación segura en internet
+ MQTT: protocolo de mensajería ligero orientado a IoT (https://mqtt.org/)
+ OSC (Open Sound Control): protocolo para comunicación en tiempo real entre dispositivos multimedia

MQTT funciona mediante un sistema de publicación y suscripción, permitiendo que múltiples dispositivos reciban información simultáneamente.

+ Interacciones inalámbricas

El curso aborda sistemas donde dispositivos se comunican en red en tiempo real, dentro del contexto de IoT.
